
# 로또 번호 생성기

## 개요

이 프로젝트는 무작위 로또 번호를 생성하는 간단한 웹 애플리케이션입니다.

## 기능

*   1부터 45까지의 숫자 중에서 6개의 고유한 무작위 번호를 생성합니다.
*   생성된 번호를 시각적으로 보기 좋게 표시합니다.
*   새로운 번호 세트를 생성하는 버튼이 있습니다.

## 프로젝트 구조

*   `index.html`: 메인 HTML 파일입니다.
*   `main.js`: 로또 번호 생성 로직을 담고 있는 JavaScript 파일입니다.
*   `style.css`: 애플리케이션 스타일을 위한 CSS 파일입니다.

## 현재 작업

*   애플리케이션의 기본 텍스트를 한국어로 번역합니다.
